# Physical Memory Index

Physical memory describes installed hardware, sensors, actuators, calibration values and measurement interpretation.

## Domains

```text
ftx/       ventilation aggregate and FTX-local hardware
heating/   heat pumps, brine, hot water, VVB/VVC interfaces
floor/     floor heating/cooling, shunts, pumps and valves
home/      rooms, zones and house-level comfort sensors
```

Infrastructure owns device identity and reachability. Physical domain files own sensor meaning, placement, actuator meaning, calibration constants, physical constraints and risks.
