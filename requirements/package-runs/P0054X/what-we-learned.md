# P0054X what we learned

1. P0054W data is valuable but only for the eSett load-profile/profilled component.
2. The operator's expected missing majority is real: the profiled/load-profile component averages about 23% of ENTSO-E SE3 actual total load in the overlap.
3. Public eSett Open Data does not appear to expose `metered` consumption per MGA; it exposes that component at MBA/SE3 level.
4. A full SE3 MGA taxonomy must wait for a monthly-read/measured per-MGA source or be explicitly scoped to profiled load only.
